# 📰 WorkNewsHub

WorkNewsHub é um projeto de portal de notícias desenvolvido com React, focado em tecnologia, carreira e trabalho remoto.

O objetivo do projeto é simular um blog moderno com categorias, busca, páginas institucionais e navegação dinâmica.

---

## 🚀 Funcionalidades

- 📌 Lista de posts dinâmicos
- ⭐ Sistema de posts em destaque (Featured)
- 🗂️ Categorias dinâmicas (Tech, Career, Home Office)
- 🔎 Sistema de busca com página dedicada
- 📄 Página individual de post
- 🧭 Navegação completa com React Router
- 📱 Layout responsivo (mobile e desktop)

---

## 📄 Páginas institucionais

- Sobre nós (`/about`)
- Política de Privacidade (`/privacy`)
- Termos de Uso (`/terms`)
- Contato (`/contact`)

Todas com layout padronizado e estrutura visual consistente.

---

## 🛠️ Tecnologias utilizadas

- React
- React Router DOM
- JavaScript (ES6+)
- CSS inline (em refatoração futura para styled components ou CSS modules)

---

## 🔎 Busca

O sistema de busca permite pesquisar posts por título, descrição e categoria, redirecionando o usuário para uma página dedicada de resultados.
